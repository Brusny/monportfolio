# Prédiction d’acceptation de crédit

Projet de classification permettant de prédire si un crédit doit être accepté.

## Étapes du projet

- Prétraitement des données
- Feature selection
- Entraînement de plusieurs modèles
- Optimisation avec GridSearchCV
- Évaluation des performances

## Modèles testés

- Logistic Regression
- SVM
- Random Forest
- XGBoost

## Technologies

- Python
- Pandas
- Scikit-learn
